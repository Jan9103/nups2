# Nushell plugin for nups2 (planetside2 file-format support)

Commands:
* `hash ps2crc64`: apply the ps2 flavour of the crc64 hashing algorhytm
* `pack2 ls`: list contents of a `.pack2` file
* `pack2 extract`: extract contents of a `.pack2` file
* `pack2 scrape_filenames`: generate a filename-list-file by scraping the contents of a `.pack2` file

Plugin version | Nushell version
-------------- | ---------------
`0.101.0` | `0.101.0`
